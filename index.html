PrivateChat-Socket.IO

A simple web application that offers public and private chat. It was implemented using Node.js and Socket.io. Also twitter bootstrap for interface and jQuery were used.

Features

  • It supports private chat beside public chat.

New user is connected

General message

  • It includes a notification system. If a message is came from a different user than the actively talked user, messages count is displayed on tab.

Sample notification

  • It includes a status system.
    • When a user is typing "User is typing" message is displayed in other users window.

User is typing

   - After a user see a message , "Seen at [***]" message is displayed.

Seen at [***]" typed notification

Requires

  • node.js - You can download it from http://nodejs.org/download/
  • socket.io

npm install socket.io
